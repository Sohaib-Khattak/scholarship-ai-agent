STEP 1: Deploy Frontend to Vercel (2 minutes)
   → Go to: https://vercel.com
   → Login with GitHub
   → Import: scholarship-ai-agent-
   → Set Root Directory: frontend
   → Add env: NEXT_PUBLIC_API_URL

STEP 2: Deploy Backend to Railway (2 minutes)
   → Go to: https://railway.app
   → Login with GitHub
   → Deploy from GitHub repo
   → Set Root Directory: backend
   → Add 9 environment variables
   → Deploy!

STEP 3: Setup Database on Supabase (1 minute)
   → Go to: https://supabase.com
   → Create new project
   → Run schema.sql in SQL Editor
   → Copy credentials to Railway

STEP 4: Connect Frontend to Backend (1 minute)
   → Update NEXT_PUBLIC_API_URL in Vercel
   → Redeploy frontend
   → Test connection

STEP 5: Test Everything (2 minutes)
   → Visit frontend URL
   → Click "Manual Scrape"
   → Approve a scholarship
   → Publish to WhatsApp
