## 🚀 QUICK START (90 Minutes)

### Phase 1: Preparation (5 min)
```
Get API Keys:
- OpenAI: https://platform.openai.com/api/keys
- Supabase: https://supabase.com
- WhatsApp: https://developers.facebook.com
```

### Phase 2: Local Setup (30 min)
```bash
# Backend
cd backend && npm install && cp .env.example .env

# Frontend
cd frontend && npm install
echo "NEXT_PUBLIC_API_URL=http://localhost:5000/api" > .env.local

# Database
# Execute schema.sql in Supabase SQL Editor
```

### Phase 3: Run Locally (5 min)
```bash
# Terminal 1: Backend
cd backend && npm run dev

# Terminal 2: Frontend
cd frontend && npm run dev

# Visit: http://localhost:3000
```

### Phase 4: Test (10 min)
```
1. Click "Manual Scrape"
2. Wait 2-3 minutes
3. Review pending scholarships
4. Approve and publish one
5. Check WhatsApp group
```

### Phase 5: Deploy (45 min)
```
1. Push to GitHub
2. Deploy frontend to Vercel
3. Deploy backend to Railway
4. Set environment variables
5. Test production system
```
